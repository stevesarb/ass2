To compile the program:
    1. Set working directory to the directory this README.txt file is stored in
    2. In the command line, enter: gcc ./code/*.c -o movies_by_year
        a. This will compile all of the .c files together and create an executable called movies_by_year
    
To run the executable:
    1. Set the working directory to the directory that contains the executable named movies_by_year
    2. In the command line, enter: ./movies_by_year
